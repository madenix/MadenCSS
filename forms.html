🔠 Inputs, Textareas & Selects












✅ Features:

🔹Full-width by default
🔹Rounded corners
🔹Smooth transitions on focus
🔹Theme-aware background and text colors

✨ Focus State
When the input is focused:

🔹 Border color changes to --primary

🔹 Light box shadow appears for accessibility

input:focus {
border-color: var(--primary);
box-shadow: 0 0 0 2px rgba(240, 187, 12, 0.3);
}

❌ Validation & Error Handling
Inputs with :invalid or error feedback can use .form-error:

Please enter a valid email.



Invalid inputs automatically receive a red (--danger) border via:
input:invalid {
border-color: var(--danger);
}


🛈 Helper Text
Use .form-help for small guidance below fields:

We'll never share your email.

💡 Notes
All form controls are 100% width by default — use a grid or layout utility to control sizing. Works best when combined with spacing.css for margins/padding and typography.css for text scale. Use semantic HTML (<label>, <input>, <textarea>, <select>) for accessibility and screen reader support.
