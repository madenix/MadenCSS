MadenCSS provides positioning utilities that allow precise control over element placement. These utilities are designed to be predictable, composable, and easy to reason about when building complex layouts.

1. Position Utilities 


.relative { position: relative; }
.absolute { position: absolute; }
.fixed    { position: fixed; }
.sticky   { position: sticky; }

.top-0    { top: 0; }
.right-0  { right: 0; }
.bottom-0 { bottom: 0; }
.left-0   { left: 0; }

.translate-top-right {top: 0; right: 0; transform: translate(50%, -50%);}
.translate-top-left {top: 0; left: 0; transform: translate(-50%, -50%);}
.translate-bottom-right {bottom: 0; right: 0; transform: translate(50%, 50%);}
.translate-bottom-left {bottom: 0; left: 0;transform: translate(-50%, 50%);}
.translate-center {top: 50%; left: 50%;transform: translate(-50%, -50%);}

Position utilities define how an element is positioned in the document flow. They are commonly used for overlays, badges, tooltips, sticky headers, and floating UI elements.

Relative

Establishes a positioning context for absolutely positioned children.

.relative

Absolute

Removes the element from normal flow and positions it relative to the nearest positioned ancestor.

.absolute

Fixed

Positions the element relative to the viewport. Commonly used for modals and floating actions.

.fixed

Sticky

Switches between relative and fixed positioning based on scroll position.

.sticky

Tip: Absolute positioning requires a parent element with .relative to ensure predictable placement.
