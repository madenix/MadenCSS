MadenCSS is a native, token-driven CSS foundation built for teams that care about long-term maintainability. It does not try to replace CSS or abstract it away. Instead, it formalizes design decisions using tokens, layers, and sensible defaults, enabling consistent UI development across products while remaining framework-agnostic and easy to extend.

Predictable Layout System

A responsive, constraint-based grid built for long-term consistency.

Token-Based Theming

Centralized design tokens enabling effortless theme customization.

Consistent Spacing Scale

A unified spacing system across margin, padding, and layout gaps.

Fluid Typography System

Responsive font sizes and weights driven by design decisions.

Layout Primitives

Flexbox and grid helpers designed as low-level building blocks.

Visual Primitives

Borders, radius, shadows, and display utilities as composable primitives.

Reference Components

Optional components demonstrating recommended structure and patterns.
