TYPOGRAPHY

 ✍️ typography.css Usage Guide

This file provides a utility-first set of classes for managing typography consistently across your project. Font weights, sizes, alignment, spacing, truncation — all covered.


🧱 Overview of Utility Classes
🔹 Font Weights
Use these to quickly apply standard weight values:

Light text (fw-light)

Bold text (fw-bold)

Extra black (fw-black)



.fw-thin: 100
.fw-light: 300
.fw-normal: 400
.fw-medium: 500
.fw-semibold: 600
.fw-bold: 700
.fw-black: 900

🔹 Text Alignment

Left aligned (text-left)

Centered text (text-center)

Right aligned (text-right)

Justified block (text-justify)


🔹 Text Transform

uppercase (text-uppercase)

LOWERCASE (text-lowercase)

capitalize each word (text-capitalize)


🔹 Font Sizes (Responsive with clamp())
Designed to scale across screen sizes.

(fs-xl) Responsive x large Heading

(fs-base) Base paragraph text


.fs-sm: 12–14px
.fs-base: 14–16px
.fs-lg: 16–20px
.fs-xl: 20–24px
.fs-xxl: 24–36px
.fs-huge: 32–48px

🔹 Truncation (truncate)
Use when you want to cut off text with ... if it overflows its container:

Very long content that won’t fit.


🔹 Word Wrapping (break-word)
Break long words instead of overflowing:

supercalifragilisticexpialidocious


✅ Tips
Use typography utilities in combination with spacing and layout classes for clean, modular components.

All sizes are responsive where possible (clamp()), no need for media queries in most cases.

Extend this file as needed with your custom type scale or font families.
