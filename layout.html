üì± .gutters

These utility classes control the spacing between columns inside a .row using the CSS gap property. The number in the class name determines the spacing size. This makes it easy to manage consistent spacing in your layout. Row gutters apply spacing in both horizontal and vertical directions between columns.

üìå Usage:
Add one of the .row-gutter-* classes to a .row element.

The number indicates the size of the horizontal and vertical spacing between columns (in rem units).
.row-gutter-0 { gap: 0; }
.row-gutter-1 { gap: 0.5rem; } /* 8px */
.row-gutter-2 { gap: 1rem; } /* 16px */
.row-gutter-3 { gap: 1.5rem; } /* 24px */
.row-gutter-4 { gap: 2rem; } /* 32px */
