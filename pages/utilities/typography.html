1. Fluid Font Sizes (clamp)

All font sizes in MadenCSS are defined using the clamp(min, fluid, max) function. This allows text to scale smoothly between minimum and maximum sizes depending on the viewport width.

2. How clamp() Works

The clamp() function defines a range in which the font size can grow or shrink:

  • Minimum: Small screens never go below this size
  • Fluid value: Scales based on viewport width
  • Maximum: Large screens never exceed this size

/* maden-config.css */
/* fs-base: clamp(0.875rem, 1vw, 1rem) */
--fs-base-min: 0.875rem;
--fs-base-fluid: 1vw;
--fs-base-max: 1rem;

/* core.css */
.fs-base {
  .fs-base  { font-size: clamp(var(--fs-base-min), var(--fs-base-fluid), var(--fs-base-max)); }
}

5. Text Helpers

Text helpers are designed to solve two very common UI problems: overflowing text and unbreakable long words. These utilities keep layouts clean without writing custom CSS.

5.1 .truncate

The .truncate utility forces text to stay on a single line. If the content is longer than the container, it is visually cut and replaced with an ellipsis (...).

This is especially useful for:

  • ‚Ä¢ Card titles
  • ‚Ä¢ Table cells
  • ‚Ä¢ Navigation labels
  • ‚Ä¢ User-generated content with unknown length

This text will be cut with an ellipsis if it overflows 


<p class="truncate">
  This text will be cut with an ellipsis if it overflows
</p>

5.2 .break-word

The .break-word utility allows extremely long words to break and wrap onto the next line instead of overflowing their container.

This solves a common problem when dealing with:

  • ‚Ä¢ Long URLs
  • ‚Ä¢ Hashes or tokens
  • ‚Ä¢ Usernames or IDs
  • ‚Ä¢ Technical strings
