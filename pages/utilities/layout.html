1. Core Layout Utilities

Page Centering and Maximum Width

These utilities are used to horizontally center your content and prevent it from stretching too wide on large screens.

Page centered using .max-width-md and .page-center

All content is constrained within --max-width-lg (default). 


<!-- Values are sourced from maden-config.css -->
.max-width-lg { max-width: var(--max-width-lg); }
.page-center  { margin-left: auto; margin-right: auto; }

<!-- Usage -->
<div class="max-width-lg page-center">
    Main page content goes here.
</div>

3. Responsive Behavior (Breakpoint: 599px)

The grid system automatically applies an **auto-stacking** rule on all screens **599px and below**, ensuring mobile-friendly layouts by default.

No breakpoint-specific column classes are required.

  • Large Screens (> 599px): Columns preserve their defined .col-* widths.
  • Mobile Screens (‚â§ 599px): The grid switches to a single-column layout, and all columns expand to full width (100%).

Disabling Mobile Stacking (Opt-out)

If you want a grid to retain its 12-column structure even on mobile devices, add the .not-responsive class to the .grid container: 


<!-- This grid remains 4 / 8 columns even on mobile -->
<div class="grid grid-gap-2 not-responsive">
    <div class="col-4">...</div> 
    <div class="col-8">...</div> 
</div>
