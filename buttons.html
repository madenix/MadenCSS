Buttons are essential interactive elements in any user interface. MadenCSS buttons are designed to be flexible, accessible, and easy to style, allowing you to create a consistent look across your app or website.

üîπ Basic Button Class: .btn

The .btn class provides foundational styles such as padding, font, background, border, and cursor behavior.

üîπ Variants (Color and Style)

üîπ Sizes

Control button size using .btn-sm (small), .btn-lg (large), and .btn-block (full width).

üîπ Border Radius (Rounded Corners)

Add rounded corners with;

.radius-sm { border-radius: 2px; }
.radius { border-radius: 4px; } /* default */
.radius-md { border-radius: 6px; }
.radius-lg { border-radius: 8px; }
.radius-xl { border-radius: 12px; }
.radius-2xl { border-radius: 16px; }
.radius-full { border-radius: 9999px; } /* for pills, circles */
üîπ Disabled State

Use the disabled attribute to prevent interaction and visually indicate inactive buttons.

<button class="btn btn-primary" disabled>Disabled Button</button>
üîπ Icon Buttons

MadenCSS allows you to include SVG icons inside your buttons with consistent sizing and color using the .btn-icon utility classes.

To insert an icon inside a button, wrap the <svg> with the class .btn-icon. The icon inherits the button text color and scales with font size.

üîπ Usage Tips & Best Practices
  • Use color variants consistently to indicate action types.
  • Avoid very small buttons for better usability on touch devices.
  • Ensure sufficient color contrast for accessibility.
  • Use btn-block for full-width buttons on small screens.
  • Include visible focus styles to support keyboard navigation.
üîπ Summary

Combine the .btn base class with variant, size, radius, and state classes to create a wide range of buttons tailored to your app‚Äôs design and functional needs.
