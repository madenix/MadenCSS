Dynamic dialogs powered by the "Checkbox Hack". These components require zero JavaScript to function.

modal.css must be included to use this component.

Basic Usage

A standard modal window. Ensure the input is placed directly before the .modal element.

Hello, MadenCSS!

This modal is controlled entirely via CSS using the + sibling selector.

<!--Basic Modal -->
<div class="p-4 bg-surface text-on-surface radius-md mb-3">
  <label for="modal-demo" class="btn btn-primary p-2">Open Basic Modal</label>
  <input type="checkbox" id="modal-demo" class="modalState" />
  
  <div class="modal">
    <label for="modal-demo" class="modalBackground"></label>
    
    <div class="modalInner bg-surface p-3 radius-lg shadow-strong">
      <div class="flex flex-justify-between">
        <span class="fs-xxl fw-bold mb-3">Hello, MadenCSS!</span>
        <label for="modal-demo" class="fs-xl px-2 fw-bold" style="cursor: pointer;">X</label>
      </div>
      <p class="mb-4">This modal is controlled entirely via CSS using the <code>+</code> sibling selector.</p>
      
      <div class="flex flex-justify-end">
        <label for="modal-demo" class="btn btn-dark p-2">Close Modal</label>
      </div>
    </div>
  </div>
</div>

Nested Modals

To create nested modals, simply place a new modalState and modal pair inside the modalInner of the first one.

Level 1

Click below to open a second modal layer.

Level 2

Stacked perfectly using independent states!

<!-- Nested Modals Layered System -->
<div class="p-4 border border-light radius-md mb-3">
  <label for="nested-1" class="btn btn-success p-2">Open Level 1</label>
  <input type="checkbox" id="nested-1" class="modalState" />

  <div class="modal">
    <label for="nested-1" class="modalBackground"></label>
    <div class="modalInner bg-surface p-3 radius-lg shadow-strong">
      <div class="flex flex-justify-between">
        <span class="fs-xxl fw-bold">Level 1</span>
        <label for="nested-1" class="fs-xl px-2 fw-bold" style="cursor: pointer;">X</label>
      </div>
      <p class="mb-4">Click below to open a second modal layer.</p>
      
      <label for="nested-2" class="btn btn-primary p-2">Open Level 2</label>
      <input type="checkbox" id="nested-2" class="modalState" />
      
      <div class="modal">
        <label for="nested-2" class="modalBackground"></label>
        <div class="modalInner bg-surface p-3 radius-lg shadow-strong border border-primary">
          <h4 class="fs-xxl fw-bold text-primary mb-2">Level 2</h4>
          <p class="mb-4">Stacked perfectly using independent states!</p>
          <label for="nested-2" class="btn btn-danger btn-sm">Close</label>
        </div>
      </div>
    </div>
  </div>
</div>
