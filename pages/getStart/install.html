Installation

MadenCSS is designed to be adopted, not installed. You can include it directly in your project using a CDN for quick experiments, or integrate the source files locally as a foundation for your own design system.

1. CDN (Quick Start)

Use the CDN to explore MadenCSS or prototype quickly. This option requires no build tools and works with plain HTML. 

<link rel="stylesheet" href="https://cdn.madencss.com/maden.css">

2. Local Integration

For long-term projects, download the source files and include them directly. This approach allows you to customize tokens, extend components, and evolve MadenCSS as part of your own design system. 

<link rel="stylesheet" href="/css/maden.css">
