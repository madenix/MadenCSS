MadenCSS provides a simple and consistent sizing system for controlling element width and height. These utilities are designed for UI components, layout sections, and proportional sizing without relying on custom CSS.

1. Max Width Utilities

Max-width utilities are commonly used to constrain page content and improve readability on large screens.

.max-width-2xs (300px)
.max-width-xs (450px)
.max-width-sm (767px)
.max-width-md (900px)
.max-width-lg (1200px)
.max-width-xl (1600px)

<div class="max-width-md page-center">
    Page content
</div>

2. Fixed Width & Height Scale

These utilities follow a predefined scale and are ideal for UI elements such as cards, buttons, avatars, and placeholders. 


<div class="width-xsm height-xsm"></div>
<div class="width-sm height-sm"></div>
<div class="width-md height-md"></div>
<div class="width-lg height-lg"></div>
<div class="width-xl height-xl"></div>
<div class="width-2xl height-2xl"></div>
<div class="width-3xl height-3xl"></div>

3. Fractional Widths

Fractional width utilities allow proportional layouts without using grid or flex column systems. They are especially useful for simple horizontal layouts.

.width-1/2
.width-1/3
.width-2/3
.width-1/4
.width-3/4
.width-1/5
.width-2/5
.width-3/5
.width-4/5
.width-1/6
.width-5/6

4. Height Utilities

Layout Heights

These utilities are commonly used for full-screen sections and page layouts. 


<section class="height-screen">Full screen section</section>
<section class="min-height-screen">At least full height</section>
