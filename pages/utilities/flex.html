MadenCSS provides a robust set of 12-column **Flex-based Grid** and alignment utilities. While the standard grid is ideal for page structures, Flex utilities excel at component-level layouts and complex alignments.

1. Flex Column System

Unlike the standard grid, .flex-col-* classes allow for precise control within a flex container. These columns also feature automatic mobile stacking at **599px**.

Flex Container (.flex .flex-wrap)

.flex-col-8
.flex-col-4
.flex-col-3
.flex-col-9

<!-- Using Flex Columns with Wrap and Gaps -->
<div class="flex flex-wrap">
    <div class="flex-col-8"> Main Content </div>
    <div class="flex-col-4"> Sidebar </div>
</div>

3. Dynamic Flex Behavior

Dynamic Flex utilities allow elements to grow, shrink, or remain fixed based on available space. This makes Flex ideal for toolbars, navigation bars, cards, and component-level layouts.

3.1 Flexible Growth (.flex-1)

Use .flex-1 to make an element fill all remaining available space inside a flex container.

Logo
Flexible Content (.flex-1)
Button
    
    <div class="flex">
        <div>Logo</div>
        <div class="flex-1">Flexible Content</div>
        <div>Button</div>
    </div>

3.2 Fixed vs Flexible Items (.flex-none)

Elements marked with .flex-none will never grow or shrink. This is useful for icons, avatars, and fixed-size buttons.

ICON
This content grows to fill the remaining space 
    
    <div class="flex">
        <div class="flex-none">ICON</div>
        <div class="flex-1">Growing Content</div>
    </div>

3.3 Responsive Control (.not-responsive)

By default, .flex-col-* elements stack vertically on mobile. Use .not-responsive to prevent stacking.

Stays Side-by-Side
Even on Mobile 
    
    <div class="flex not-responsive">
        <div class="flex-col-6">A</div>
        <div class="flex-col-6">B</div>
    </div>

3.4 Direction Control (.flex-col / .flex-row)

Control the flow direction of flex items using .flex-col or .flex-row.

Header
Content
Footer

<div class="flex flex-col">
    <div>Header</div>
    <div>Content</div>
    <div>Footer</div>
</div>
