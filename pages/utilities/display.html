MadenCSS provides simple and predictable utility classes to control element visibility, layout behavior, and visual emphasis. These utilities allow you to show, hide, or subtly fade elements without writing custom CSS.

1. Display Utilities

Display utilities control how an element participates in the layout flow. They are commonly used for toggling visibility, inline behavior, or layout adjustments.

Block

The element takes the full width and starts on a new line.

.block

Inline

The element flows inline with text and ignores width/height.

.inline

Inline Block

Inline behavior with width and height support.

.inline-block

Hidden

Completely removes the element from layout and visibility.

.hidden 

<div class="block">Block element</div>
<span class="inline">Inline text</span>
<div class="inline-block">Inline block</div>
<div class="hidden">This will not be visible</div>

Tip: Use .hidden for conditional visibility instead of removing elements from the DOM.

2. Opacity Utilities

Opacity utilities allow you to visually de-emphasize elements, create disabled states, or layer content without affecting layout.

opacity-25

opacity-50

opacity-75

opacity-100

opacity-0


<button class="opacity-50">Disabled-like button</button>
<p class="opacity-75">Secondary text</p>
<div class="opacity-0">Invisible but still in layout</div>

Note: Unlike .hidden, opacity utilities do not remove elements from the layout. The element remains interactive unless combined with other utilities.
