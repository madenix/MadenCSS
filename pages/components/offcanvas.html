The Offcanvas component allows you to build hidden sidebars for navigation, shopping carts, or settings that slide in from any edge of the screen. Built on the "Checkbox Hack" principle, it provides a pure CSS solution without requiring any JavaScript.

This component uses the .offcanvas-toggle (hidden checkbox) to manage state. The .offcanvas-backdrop provides a dimmed background that closes the panel when clicked.

1. Left Direction (Default)

The most common usage for mobile navigation menus.
