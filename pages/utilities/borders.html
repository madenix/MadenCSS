MadenCSS offers semantic and utility-first classes for borders, corner rounding (radius), and shadow effects. This structure allows you to create consistent and scalable user interfaces.

2. Borders

MadenCSS provides atomic and semantic utility classes to control border thickness and direction. Border weights are centrally defined in the maden-config.css file.

Border Thicknesses

border-thin
border-normal
border-thick
border-heavy
<div class="border-full border-thick p-4">Thick Border</div>

Border Directions

Borders can be applied to specific directions only. This allows you to create more controlled dividers and layout boundaries.

border-t
border-r
border-b
border-l
<div class="border-b border-thin p-4">Bottom Border Only</div>

Removing Borders

The border-none class completely removes all borders from an element. 

<div class="border-0 p-4">No Border</div>

Note: To change border thickness values, it is recommended to update the CSS variables (--border-thin, --border-normal, etc.) directly.
