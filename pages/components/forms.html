1. Base layer forms

By default forms are styled with in MadenCss base layer. Thanks to the @layer design, existing form appearence can be safely extended or overridden without breaking the core system files.

üëâBase layer default values can be found below. 

  input, button, textarea, select {
      background-color: var(--surface);
      color: var(--on-surface);
      padding: var(--spacing-2) var(--spacing-3);
      width: 100%;
      border: var(--border-thin) solid var(--mid);
      border-radius: var(--radius);
      transition: border-color var(--focus-transition), box-shadow var(--focus-transition);
      font: inherit;
   }

   input:focus, button:focus, textarea:focus, select:focus {
      background-color: var(--focus-background);
      color: var(--on-focus);
      border-color: var(--focus-border-color);
      outline: var(--focus-border-thickness) solid var(--focus-border-color);
      outline-offset: var(--focus-outline-offset);
   }

   input:disabled, textarea:disabled, select:disabled {
      opacity: 0.6;
      cursor: not-allowed;
   }
  
   input::placeholder {transition: .2s;}
   input:focus::placeholder {opacity: 0;}
   ::placeholder { opacity: 0.7; }

MadenCSS provides a robust base layer that facilitates the consistent design of standard form interfaces.

3. Range, Checkbox, Radio Inputs (Accent Utilities)

MadenCSS styles native HTML form controls without breaking their default behavior. Range, Checkbox, Radio elements are styled using accent utility classes, which rely on the native accent-color CSS property.

This approach keeps range inputs fully native, accessible, and theme-aware, while allowing quick visual customization through semantic color tokens.

How It Works

Each accent-* utility maps directly to a design token:

  • accent-primary ‚Üí --primary
  • accent-success ‚Üí --success
  • accent-warning ‚Üí --warning
  • accent-danger ‚Üí --danger
  • accent-info ‚Üí --info
  • accent-dark ‚Üí --dark
  • accent-mid ‚Üí --mid
  • accent-light ‚Üí --light

Range Variants

All semantic colors defined in maden-config.css can be used as accent colors for range inputs.
