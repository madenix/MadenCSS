Sticky Behavior

Remains fixed at the top of the viewport on desktop screens even when content is scrolled.

Smart Backdrop

No JavaScript

Developed entirely using the Checkbox Hack technique for maximum performance.

Class Description
.sticky-sidebar Fixes the menu to the top using sticky positioning. If omitted, a static menu is rendered.
.sidebar-toggle-btn The button (Label) that toggles the menu on mobile. Automatically hidden on desktop.
.sidebar-backdrop A layer that darkens the background on mobile and closes the menu upon clicking.
.sidebar-menu The flexible vertical area containing navigation links and content.
