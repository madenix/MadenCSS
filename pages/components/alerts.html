⚠️ Alerts

Components used to display information, success, or error messages to the user.

alert.css must be included to use these components.

Basic Usage

This is a primary information alert.
Action completed successfully!
An error occurred, please try again.
Attention! This action cannot be undone.
Informational alert!
<div class="alert alert-primary">...</div>
<div class="alert alert-success">...</div>
<div class="alert alert-danger">...</div>
<div class="alert alert-warning">...</div>

Outline Alerts

Add the alert-outline class for alerts with a transparent background and colored borders.

Outline primary alert.
Outline success message.
Outline warning message.
Outline danger message.
Outline info message.
<div class="alert alert-outline alert-primary">Outline primary alert.</div>
<div class="alert alert-outline alert-success">Outline success message.</div>
<div class="alert alert-outline alert-warning">Outline warning message.</div>
<div class="alert alert-outline alert-danger">Outline danger message.</div>

MadenCSS Power: Utility Customization

Thanks to the Cascade Layers (@layer) structure, you can instantly modify alert components with utility classes without writing any custom CSS.

Customized Alert (Shadow, full radius, and wide horizontal padding) 
<!-- shadow-strong, radius-full, and px-8 classes override the component styles instantly -->
<div class="alert alert-warning shadow-strong radius-full px-8">
  Customized Alert
</div>
