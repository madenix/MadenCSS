Spacing utilities in MadenCSS are built on a consistent design scale powered by CSS variables. Instead of arbitrary values, all margin and padding classes are derived from a single base unit to ensure visual harmony across layouts.

1. Spacing Scale

All spacing values are calculated from a base unit of 4px. This scale is defined in maden-config.css and can be customized globally.

  • 0 ‚Üí 0
  • 1 ‚Üí 4px
  • 2 ‚Üí 8px
  • 3 ‚Üí 16px
  • 4 ‚Üí 24px
  • 5 ‚Üí 32px
  • 6 ‚Üí 40px
  • 7 ‚Üí 48px
  • 8 ‚Üí 64px

--spacing-unit: 4px;
--spacing-3: calc(var(--spacing-unit) * 4); /* 16px */

2. Margin Utilities

Margin utilities control the outer spacing of elements. Classes follow a simple and predictable naming convention.

.m-* ‚Äî All sides
.mt-*, .mr-*, .mb-*, .ml-* ‚Äî Single side
.mx-* ‚Äî Horizontal
.my-* ‚Äî Vertical 

<div class="mt-4 mb-6">Spacing example</div>
<div class="mx-3">Horizontal margin</div>

3. Padding Utilities

Padding utilities control the inner spacing of elements and use the same scale as margin utilities.

.p-* ‚Äî All sides
.pt-*, .pr-*, .pb-*, .pl-* ‚Äî Single side
.px-* ‚Äî Horizontal
.py-* ‚Äî Vertical 

<div class="p-5">Padded container</div>
<button class="px-4 py-2">Button</button>
