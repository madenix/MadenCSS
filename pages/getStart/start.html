Get Started With MADENCSS

MadenCSS provides a minimal CSS foundation that lets you build layouts and design systems without locking you into a framework.

1. Create a basic HTML file

Make sure <meta name="viewport"> is included for proper mobile responsiveness. 

<!doctype html>
<html lang="en" data-theme="dark">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>MadenCss demo</title>
</head>
<body>
  <h1>Hello, world!</h1>
</body>
</html>

2. Include MadenCSS base files

Include maden-config.css (configuration) and core.css (utility layer) inside the <head>. 

<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>MadenCss demo</title>

  <!-- Link required CSS files -->
  <link rel="stylesheet" href="maden-config.css">
  <link rel="stylesheet" href="core.css">
</head>

3. Build your first layout

Start by applying layout helper classes to the body element.

bg-background, text-on-background, max-width-lg, page-center 

<body class="bg-background text-on-background max-width-lg page-center">

Now replace the content with a simple page layout using the 12-column grid. 

<div class="grid p-4 grid-gap-2">

  <div class="col-12 bg-surface p-4 shadow-light radius-md">
    NAVBAR
  </div>

  <div class="col-3 bg-surface p-4 shadow-light radius-md">
    SIDEBAR
  </div>

  <div class="col-9 bg-surface p-4 shadow-light radius-md">
    CONTENT
  </div>

  <div class="col-12 bg-surface p-4 shadow-light radius-md">
    FOOTER
  </div>

</div>

Grid is used to define the page structure. Content alignment inside sections can be handled with flex utilities. 

<div class="flex">
  <div class="flex-col-3">LOGO</div>
  <div class="flex-col-9">
    HOME | ABOUT | CONTACT
  </div>
</div>

MadenCSS is utility-first and layout-driven. Components are optional, not mandatory. You control the structure.

4. Final Page Example

Until now, we have built the page step by step by learning MadenCSS core concepts. To make everything clearer, below is a complete and final example that brings all these pieces together.

This example demonstrates a real-world page layout using MadenCSS grid and flex utilities, including a navbar, sidebar, content area and footer. You can use this layout as a starting point for your own projects.

<html lang="en" data-theme="dark">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">

  </head>

  <body class="bg-background text-on-background max-width-lg page-center">

    <!-- Page Layout -->
    <div class="grid p-4 grid-gap-2">

      <!-- Navbar -->
      <div class="col-12 bg-surface text-on-surface p-4 shadow-light radius-md">
        <div class="flex">
          <div class="flex-col-3 fw-bold">LOGO</div>
          <div class="flex-col-9 text-center">
            <a href="" class="p-2">HOME</a> |
            <a href="" class="p-2">ABOUT</a> |
            <a href="" class="p-2">CONTACT</a>
          </div>
        </div>
      </div>

      <!-- Sidebar -->
      <div class="col-3 bg-surface text-on-surface p-4 shadow-light radius-md">
        SIDEBAR
      </div>

      <!-- Content -->
      <div class="col-9 bg-surface text-on-surface p-4 shadow-light radius-md">
        <h2 class="mb-2">Content Area</h2>
        <p>
          This is the main content area. Layout is handled by the grid system,
          while internal alignment can use flex utilities.
          <br/><br/><br/><br/>
        </p>
      </div>

      <!-- Footer -->
      <div class="col-12 bg-surface text-on-surface p-4 shadow-light radius-md text-center">
        FOOTER
      </div>

    </div>
    <!-- Page Layout -->

  </body>
</html>

As you can see, MadenCSS allows you to create clean and predictable layouts without writing custom CSS. By combining grid and flex utilities, you can build scalable page structures with minimal effort.
