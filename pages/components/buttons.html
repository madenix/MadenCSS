▶️ Buttons

Standard button components for actions in forms, dialogs, and more. Highly flexible and utility-ready.

buttons.css must be included to use these components.

Basic Usage (Solid)

Standard buttons using theme-defined semantic colors.

<button class="btn btn-primary">Primary</button>
<button class="btn btn-success">Success</button>
<button class="btn btn-danger">Danger</button>
<button class="btn btn-warning">Warning</button>
<button class="btn btn-info">Info</button>
<button class="btn btn-dark">Dark</button>
<button class="btn btn-light">Light</button>

Outline Buttons

Use btn-outline combined with btn-outline-{color} for a lighter visual style.


<button class="btn btn-outline btn-outline-primary">Primary</button>
<button class="btn btn-outline btn-outline-success">Success</button>
<button class="btn btn-outline btn-outline-danger">Danger</button>
<button class="btn btn-outline btn-outline-warning">Warning</button>
<button class="btn btn-outline btn-outline-info">Info</button>
<button class="btn btn-outline btn-outline-dark">Dark</button>
<button class="btn btn-outline btn-outline-light">Light</button>

Sizes

Different sizes for different contexts using btn-sm or btn-lg.

<button class="btn btn-primary btn-sm">Small</button>
<button class="btn btn-primary">Default</button>
<button class="btn btn-primary btn-lg">Large</button>

MadenCSS Power: Instant Customization

Since buttons are defined in the components layer, you can override any property using utility classes without writing custom CSS.

<!-- Override anything: radius-full, shadow-strong, px-8, radius-0, border-thick -->
<button class="btn btn-success radius-full shadow-strong px-8">Custom</button>
<button class="btn btn-outline btn-outline-primary radius-0 border-thick">Square</button>
