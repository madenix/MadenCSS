MadenCSS utilizes CSS Variables and the HTML data-theme attribute to enable fast and fluid switching between Dark and Light themes. This approach provides the flexibility to manage the theme with minimal JavaScript and easily supports native browser theme preferences.

1. Theme Mechanism

All color variables are defined in the maden-config.css file within two main blocks:

  • :root **Block:** Defines the color variables for the **Dark Mode** (the default theme of MadenCSS).
  • html[data-theme="light"] **Block:** Contains the **Light Mode** colors which override the Dark Mode variables when the data-theme attribute is set to "light".

By simply toggling the attribute on the html tag, you can instantaneously update the colors of the entire interface. No additional CSS files are needed for theme switching.

2. HTML Structure

You can set the default theme of your page using the data-theme attribute on the <html> tag:

Dark Mode


<html lang="en" data-theme="dark"> 
    <body class="bg-background text-on-background">...</body>
</html>

Light Mode


<html lang="en" data-theme="light"> 
    <body class="bg-background text-on-background">...</body>
</html>

3. Theme Toggling (Optional JavaScript)

To switch the theme based on user interaction, we recommend using a minimal JavaScript snippet to toggle the data-theme attribute between Dark and Light: 


function toggleTheme() {
    const htmlElement = document.documentElement;
    const currentTheme = htmlElement.getAttribute('data-theme');
    const newTheme = (currentTheme === 'dark') ? 'light' : 'dark';

    htmlElement.setAttribute('data-theme', newTheme);
    
    // Optional: Use localStorage to remember the user's preference
    localStorage.setItem('theme', newTheme);
}

// <button onclick="toggleTheme()">Toggle Theme</button>
