MadenCSS provides simple, atomic color classes designed to quickly style your interface's background and text color. All these classes use the CSS variables (--primary, --surface, etc.) that you defined in the maden-config.css file.

1. Semantic Colors (Purpose-Driven)

These colors represent meaning within an interface (such as success, warning, or error). They can be used for both background (bg-*) and text color (text-on-*). Text colors are automatically designed to be high-contrast against their corresponding background.

Primary

Main accent, buttons, links.

Classes:
bg-primary text-on-primary

Success

Successful operations, confirmations.

Classes:
bg-success text-on-success

Danger

Errors, critical actions.

Classes:
bg-danger text-on-danger

Warning

Important alerts, attention grabbing.

Classes:
bg-warning text-on-warning

Info

Informational notes.

Classes:
bg-info text-on-info

2. Surface and Neutral Colors (Gray Tones & Surfaces)

These classes are used for the general layout of the page, backgrounds, cards, and grayscale elements.

Background

The main page background.

Classes:
bg-background text-on-background

Surface

Cards, panels, components.

Classes:
bg-surface text-on-surface

Mid

Secondary text, dividing lines.

Classes:
bg-mid text-on-mid

Light

Light gray, form fields (in Light Theme).

Classes:
bg-light text-on-light

Dark

Dark gray, form fields (in Dark Theme).

Classes:
bg-dark text-on-dark

3. Usage Examples

Use the bg-* utility class to set the background color, and the text-* class to set the text color: 


<!-- Set the background color to Primary -->
<div class="p-4 bg-primary text-on-primary radius">
    This is a box with the Primary color.
</div>

<!-- Set the text color to Danger -->
<p class="fs-lg text-danger fw-bold">
    Error! Please check the form.
</p>

<!-- Use the Surface color -->
<div class="p-3 bg-surface text-on-surface radius shadow-light">
    This is a Card or Panel surface.
</div>
